## Task 2: `cleanup_case_dups.py` — 生成待删幻影清单

写工具:列出 9 域大写幻影路径,NUL 分隔输出到 stdout(供 git rm 消费),并提供人类可读 dry-run。

**Files:**
- Create: `cleanup_case_dups.py`

**Interfaces:**
- Produces:
  - `ghost_paths() -> list[str]` —— 返回 9 域大写幻影路径列表(排除 3DGenAI、排除磁盘真实小写域)。
  - `main()`:`--list` 人类可读打印(域分组 + 总数);`--nul` 输出 NUL 分隔路径(无尾随换行)供 `git rm --pathspec-from-file`。

- [ ] **Step 1: 写脚本**

```python
# cleanup_case_dups.py
import os
import sys
import subprocess

POSTS = "content/posts"
EXCLUDE = {"3DGenAI"}  # 留子项目 C,本次不动


def ghost_paths():
    """9 域大写幻影路径:git 跟踪、顶层域名不在磁盘(全小写)、且非 3DGenAI。"""
    disk = set(os.listdir(POSTS))  # 磁盘真实域名(全小写)
    out = subprocess.run(
        ["git", "ls-files", "-z", POSTS],
        capture_output=True, check=True,
    ).stdout.decode("utf-8")
    paths = [p for p in out.split("\0") if p]
    ghosts = []
    for p in paths:
        parts = p.split("/")
        if len(parts) < 3:
            continue
        dom = parts[2]
        if dom in disk:
            continue  # 真实小写域,保留
        if dom in EXCLUDE:
            continue  # 3DGenAI,本次不动
        ghosts.append(p)
    return ghosts


def main():
    ghosts = ghost_paths()
    if "--nul" in sys.argv:
        sys.stdout.write("\0".join(ghosts))
        return
    # 默认 --list:人类可读
    from collections import Counter
    by_dom = Counter(p.split("/")[2] for p in ghosts)
    for dom, n in sorted(by_dom.items(), key=lambda x: -x[1]):
        print("  %4d  %s" % (n, dom))
    print("=== 待删大写幻影路径总数:%d ===" % len(ghosts))


if __name__ == "__main__":
    main()
```
