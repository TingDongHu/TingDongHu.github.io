在 OpenPCDet  中，每个模型的推理结果通常是**一个包含多个键值对的字典**，其中包含与 3D 检测任务相关的信息。不同模型的输出结构可能略有不同，但一般来说，模型输出通常包含以下几个关键字段：

### 1. bbox (Bounding Boxes)
这是模型预测的 **3D 边界框**，表示检测到的物体的位置和尺寸。通常每个框会包含以下信息：

- **x, y, z**：边界框的中心坐标。
- **dx, dy, dz**：边界框的长、宽、高。
- **rotation**：边界框的旋转角度。

示例：
``` python
bbox = [x, y, z, dx, dy, dz, rotation]
```

### 2. scores (Confidence Scores)
这是每个预测框的 置信度分数，表示模型对该边界框是物体的信心。通常这是一个 [0, 1] 范围内的值，值越大表示模型认为该框内物体的可能性越高。
示例：
``` python
scores = [0.85, 0.92, 0.78]  # 每个框的置信度
```
### 3. labels (Class Labels)
这是每个边界框对应的 类别标签，表示该框内检测到的物体类型。标签是一个整数，通常从 0 开始，每个数字对应一个物体类别（例如，0 可能代表“Car”，1 代表“Pedestrian”等）。
```python
labels = [0, 1, 2]  # 类别标签
```
### 4. semantic_scores 或 semantic_labels（可选）

示例：
``` python
semantic_scores = [0.95, 0.89, 0.76]
semantic_labels = [0, 1, 2]
```
模型输出结构示例：
``` python
{
    'bbox': [[10, 20, 30, 5, 3, 2, 0.1], [50, 60, 70, 6, 4, 3, 0.2]],  # 2 个物体
    'scores': [0.9, 0.8],
    'labels': [0, 1]
}
{
    'bbox': [[12, 22, 32, 5, 3, 2, 0.05], [48, 58, 68, 6, 4, 3, 0.15]],  # 2 个物体
    'scores': [0.85, 0.75],
    'labels': [0, 1]
}
{
    'bbox': [[11, 21, 31, 5, 3, 2, 0.1], [51, 61, 71, 6, 4, 3, 0.25]],  # 2 个物体
    'scores': [0.88, 0.77],
    'labels': [0, 1]
}
```
### 总结
每个模型的输出通常包括以下内容：

- **bbox**：检测到的物体的 3D 边界框。
- **scores**：每个框的置信度分数。
- **labels**：每个框的类别标签。
